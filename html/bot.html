Discord.py bot啟動教學

在這裡，您將學到如何使用 Python 開發 Discord 機器人。

步驟 1: 準備工作

在開始之前，請確保您已經前往discord api創建了discord bot，並獲取了 Bot Token。

步驟 2: 安裝 discord.py

使用以下命令安裝 discord.py 模組：

pip install discord.py

步驟 3: 編寫機器人程式

以下是一個簡單的discordbot啟動程式，使用discord.py和client，那麼，我們開始吧!!

        import discord
        from discord.ext
        
        intents = discord.Intents.all()

        client = commands.Bot(command_prefix="%", intents=intents)
        
        @client.event
        async def on_ready():
            print('bot is online')
        
        client.run("your bot token")
