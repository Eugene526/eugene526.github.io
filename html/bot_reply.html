步驟 1: 編寫機器人程式

輸入以下的程式碼後當使用者發送'ping'時，機器人會回覆'pong'，如果想要增加更多回覆訊息，只要打上 

if message.content == '機器人偵測的訊息':
    await message.reply('回覆的訊息')
就可以了!

          @client.event
          #當有訊息時
          async def on_message(message:discord.Message):
              #排除自己的訊息，避免陷入無限循環
              if message.author == client.user:
                  return
              #如果包含 ping，機器人回傳 pong
              if message.content == 'ping':
                  await message.reply('pong')
